Standard exit codes (I)

  • 0 == OK

  • 1 == General error code

  • 2 == Misuse of shell builtin

  • 126 == Cannot execute

Standard exit codes (II)

  • 127 == Command not found

  • 128 == Invalid exit value

  • (128 + n) == Process killed with signal 'n'
